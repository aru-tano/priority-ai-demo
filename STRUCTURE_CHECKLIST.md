## ダッシュボード本体の機能

- [ ] 委任システム（担当者・回収予定日・ステータス変更）
- [ ] カレンダービュー（月間・週間切り替え）
- [ ] リストビュー（ソート・フィルタ・検索）
- [ ] ボードビュー（未着手/進行中/委任/完了の4カラム）
- [ ] ブリーフィング（iframe埋め込み）
- [ ] ダークモード
- [ ] localStorageによる状態保持
- [ ] GAS連携（オプション）

## 表現ルール

- [ ] センシティブ情報なし（障害名・病名・家庭環境）
- [ ] 担任は校長発表まで書かない
- [ ] 全職員への敬意ある表現
- [ ] 下心・戦略臭ゼロ
